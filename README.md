# BrainBolt

BrainBolt is a full-stack quiz application built using React, Node.js, and PostgreSQL.

## Features
- Displays quiz questions
- Multiple choice answers
- Shows correct answer
- Next question navigation

## Tech Stack

Frontend:
- React (Vite + TypeScript)

Backend:
- Node.js
- Express
- PostgreSQL
- Redis

## How to Run

Backend:
cd backend
npm install
npm run dev

Frontend:
cd frontend
npm install
npm run dev

Frontend runs on: http://localhost:5173  
Backend runs on: http://localhost:4000
